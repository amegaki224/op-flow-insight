# 更新履歴

[简体中文](CHANGELOG.md) | [English](CHANGELOG_EN.md) |
[日本語](CHANGELOG_JA.md)

公開リポジトリを作成する前に、複数回のローカル版更新を行いました。`r1`～`r4` は
個別のソースコミットを保存していないため、履歴バイナリアーカイブとして公開しています。
GitHub が自動生成する「Source code」は各旧版のソースではありません。`r5` 以降は
リリースタグとソースコミットが対応します。

## 0.1.1-r8

- dnsmasq DHCP リースと Linux 近隣テーブルの MAC を使い、同じ端末の IPv4／IPv6 を
  一つのホスト行へ統合。
- ホスト行で IPv4、LAN IPv6、リンクローカル IPv6、グローバル IPv6 を区別し、
  ルーター自身の LAN IPv4／IPv6 アドレスも個別に表示。
- 各ホストの現在の接続画面に IPv4／IPv6 サブタブを追加。
- アクティブ通信または有効な LAN 近隣情報を持つホストだけをオンライン一覧へ表示。
  オフライン中も ID、累積値、履歴を保持し、再接続時に同じ記録へ継続集計。
- 端末別の日次履歴を保存し、指定した日・月・四半期・年で集計して UTF-8 TXT へ出力。
- ルーターのローカル時刻で毎月 1 日 00:00 に「今月」の値をリセットし、履歴と
  アクティブ接続の基準値は保持。
- 状態 v1 から v2 へ自動移行し、従来の累積値を移行日の履歴として保存。
- 履歴／出力用 rpcd メソッド、中国語・日本語翻訳、デュアルスタック統合、
  オフライン非表示、月次切替、TXT 出力の回帰テストを追加。
- 1920、1280、480 幅で列の整列、表内スクロール、ページ横方向の非オーバーフローを検証。

## 0.1.1-r7

- 画面を LuCI 標準の `cbi-map`、`cbi-section`、`cbi-tabmenu`、`table` 構造へ
  移行し、見出し、タブ、セクション、行表示を ImmortalWrt 標準画面に統一。
- 独自ダッシュボードカードを廃止し、トラフィック概要を標準セクションと表で表示。
- 背景、境界線、行色、ダークモードを現在の LuCI テーマへ委ね、Argon を含む
  各テーマで一貫した表示を実現。
- ダウンロード青、アップロード緑、リスクバッジ、トレンドグラフ、ホストクリックによる
  現在の接続への移動を維持。
- 固定列レイアウトと表内横スクロールにより、PC の拡大縮小および狭い画面でも
  見出しとデータの整列を維持。
- 2 秒ごとのリアルタイム更新後も LuCI 標準タブの選択状態を維持。
- 2560、1920、1280、768 ダーク、480 ダークの各画面幅でブラウザー回帰テストに合格。

## 0.1.1-r6

- LuCI のソース兼フォールバック言語を英語へ変更し、全画面で `_()` 翻訳 API を使用。
- 任意の簡体字中国語パッケージ `luci-i18n-op-flow-zh-cn` を追加。
- 任意の日本語パッケージ `luci-i18n-op-flow-ja` を追加。英語 UI に言語パッケージは不要。
- バックエンドの状態警告を安定した英語メッセージへ変更し、画面側で翻訳。
- ubus から LAN インターフェースへ委任されたグローバル IPv6 プレフィックスを自動検出。
  Speedtest などが IPv6 を使用した場合にリアルタイム／累積通信量へ計上されない問題を修正し、
  現在の監視プレフィックスを画面に表示。
- CSS URL に版番号を付与し、読込完了後に再配置する処理とプラグイン限定の重要レイアウト
  保護を追加。PC ブラウザーの古い CSS キャッシュや LuCI テーマ上書きによる崩れを防止。
- 長い英語／日本語見出しの列幅と狭い画面の内部スクロールを改善し、見出しとデータの
  整列、ダウンロード青、アップロード緑を維持。
- 翻訳網羅性検査と、APK/IPK 言語パッケージのメタデータ、アーキテクチャ、LMO 内容検証を追加。
- 2560、1920、1280、768 ダーク、480 ダークの各画面幅でブラウザー回帰テストに合格。

## 0.1.1-r5

- リアルタイム帯域グラフに時間 X 軸、速度 Y 軸、自動単位を追加。
- 「リアルタイム推移／LAN ホスト／現在の接続」タブを追加。
- LAN ホストをクリックすると、そのホストの現在の接続だけを表示。
- ホストを IP 数値順に固定し、通信量が変化しても行位置が移動しないよう改善。
- ダウンロードは青、アップロードは緑という表示規則を維持。
- APK 更新フックの非ゼロ終了による誤解を招くインストールエラーを修正。
- ライト／ダークテーマと複数画面幅でレイアウトを検証。

## 0.1.1-r4

- SVG 名前空間でトレンド線を生成し、凡例だけ表示される問題を修正。
- LuCI テーマごとの右上ツールバー位置を修正。
- ホスト表の見出しとデータ列のずれを修正。
- ダウンロードを青、アップロードを緑で表示。
- ブラウザー拡大縮小時のレスポンシブ配置を改善。

## 0.1.1-r3

- ダークテーマにニュートラルグレーを採用し、強い白黒コントラストを低減。
- プラグイン CSS を分離し、LuCI テーマのグローバル規則による干渉を低減。
- テーマ、解像度、ズーム率に対応する Flex／Grid／レスポンシブ配置を追加。
- 狭い画面でカード、表、接続詳細の可読性を改善。

## 0.1.1-r2

- リアルタイムデータを表示する Flow Insight ステータス画面を追加。
- リアルタイム上下速度、累積通信量、ホスト数、接続数、最高リスクを表示。
- LAN ホスト一覧と現在の接続詳細を追加。
- ImmortalWrt 24.10.x / OpenWrt opkg 向け x86_64 IPK のビルドとインストールに対応。
