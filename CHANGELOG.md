# 更新记录

[简体中文](CHANGELOG.md) | [English](CHANGELOG_EN.md) |
[日本語](CHANGELOG_JA.md)

本项目在公开仓库建立前完成了数轮本地迭代。`r1` 至 `r4` 当时没有保留各自独立的 Git
提交，因此这些版本在 GitHub Releases 中作为历史二进制归档；GitHub 自动生成的
“Source code”压缩包不代表对应旧版本源码。自 `r5` 起，版本标签与源码提交一一对应。

## 0.1.1-r10

- 修复设备更换网段或重新获取地址后，内网主机列表仍同时显示旧 IP 和当前 IP 的问题。
- 当前地址改为综合活动 conntrack、Linux 邻居状态和首选有效 DHCP lease 判断；
  同地址族存在更强的当前证据时，不再显示仅剩 `STALE` 邻居记录的旧地址。
- 过滤只影响实时内网主机列表；旧 IP 仍保留在设备档案和流量历史中，已有累计记录不会丢失。
- DHCP lease 解析增加过期时间检查；同一 MAC 暂存多条 lease 时优先使用静态 lease
  或到期时间更晚的当前 lease。
- 新增跨网段换址回归测试，并验证仍有活动连接的地址不会被错误隐藏。

## 0.1.1-r9

- 将 dnsmasq DHCP lease 的主机名刷新周期从 30 秒缩短到约 2 秒，与默认流量采集周期一致。
- DHCP hostname 修改后会立即同步到在线主机、设备档案和保留的流量历史，无需重启插件。
- 按日、月、季度、年查询历史以及导出 TXT 时，统一使用查询或导出时设备最新的 hostname，
  不再显示流量产生时缓存的旧名称。
- 新增主机在线但没有新流量时的改名回归测试，并验证 TXT 只包含新名称。

## 0.1.1-r8

- 使用 DHCP lease 和 Linux 邻居表的 MAC 将同一设备的 IPv4、IPv6 合并为一个主机栏目。
- 在主机行中区分 IPv4、LAN IPv6、链路本地 IPv6 和公网 IPv6，并单独标出路由器
  自身的 LAN IPv4/IPv6 地址。
- 当前连接详情增加 IPv4/IPv6 子标签，避免双栈连接挤在同一张表内。
- 在线列表只显示有活动流量或有效 LAN 邻居记录的主机；离线主机立即退出实时列表，
  但累计量、身份和历史继续保留，再次上线后接续记录。
- 新增按日保存的设备流量档案，可选择指定日、月、季度、年汇总，并导出 UTF-8 TXT。
- “本月累计”按路由器本地时间在每月 1 日 00:00 自动重置；历史记录及跨月连接基线保留。
- 状态格式从 v1 自动迁移到 v2，旧累计量归档到迁移日，不丢失原有计数。
- 新增历史查询/导出 rpcd 方法、中日翻译，以及双栈、离线隐藏、跨月和导出的回归测试。
- 在 1920、1280 和 480 宽度下验证表头对齐、受控表内滚动与无页面横向溢出。

## 0.1.1-r7

- 页面改用 LuCI 原生 `cbi-map`、`cbi-section`、`cbi-tabmenu` 与 `table` 结构，
  标题、标签、栏目和表格样式与 ImmortalWrt 自带页面保持一致。
- 流量概览改为原生栏目和表格，不再维护一套独立的仪表盘卡片主题。
- 页面颜色、边框、行底色和暗色模式由当前 LuCI 主题统一控制，提升 Argon
  及其他主题下的一致性。
- 保留下载蓝色、上传绿色、风险徽标、趋势图与主机点击查看当前连接的功能语义。
- 表格采用固定列布局和受控横向滚动，在 PC 缩放与窄屏下保持表头和数据对齐。
- 原生标签选择状态可跨两秒实时刷新保持，不会因数据更新跳回第一个标签。
- 通过 2560、1920、1280、768 暗色与 480 暗色视口的浏览器回归测试。

## 0.1.1-r6

- 将 LuCI 源码默认文字改为英语，并全面接入 LuCI `_()` 翻译接口。
- 新增可选简体中文语言包 `luci-i18n-op-flow-zh-cn`。
- 新增可选日语语言包 `luci-i18n-op-flow-ja`；英语默认界面不需要语言包。
- 后台健康警告改用稳定英文消息，并在页面端按语言翻译。
- 自动通过 ubus 识别 LAN 接口的运营商委派公网 IPv6 前缀，修复 IPv6
  Speedtest 等流量未计入实时速率与累计流量的问题；页面会显示实际监测前缀。
- 为 CSS 地址加入版本标识、样式加载后重新排版，并增加仅作用于插件的关键布局保护，
  避免 PC 浏览器缓存旧样式或 LuCI 主题覆盖造成错位。
- 加强长英文/日文表头的列宽与窄屏滚动，保持表头、数据、下载蓝色和上传绿色对齐。
- 新增翻译完整性检查，并在 APK/IPK 流水线中验证语言包、架构与 LMO 内容。
- 通过 2560、1920、1280、768 暗色与 480 暗色视口的浏览器回归测试。

## 0.1.1-r5

- 保持下载蓝色、上传绿色的视觉语义。
- 修复 APK 升级钩子的非零退出问题，避免安装完成后显示误导性错误。
- 完成亮色、暗色以及多种窗口宽度下的布局验证。

## 0.1.1-r3

- 使用弹性布局、网格布局和响应式断点适配不同主题、分辨率与浏览器缩放比例。

## 0.1.1-r2

- 展示内网主机列表和当前连接详情。
