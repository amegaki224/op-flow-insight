# 更新记录

本项目在公开仓库建立前完成了数轮本地迭代。`r1` 至 `r4` 当时没有保留各自独立的 Git
提交，因此这些版本在 GitHub Releases 中作为历史二进制归档；GitHub 自动生成的
“Source code”压缩包不代表对应旧版本源码。自 `r5` 起，版本标签与源码提交一一对应。

## 0.1.1-r5

- 为实时带宽趋势图补充时间横轴、速率纵轴和自适应单位。
- 增加“实时趋势 / 内网主机 / 当前连接”标签页。
- 点击内网主机可跳转并筛选该主机的当前连接。
- 主机按 IP 数值顺序固定排列，刷新和流量变化时不再跳动。
- 保持下载蓝色、上传绿色的视觉语义。
- 修复 APK 升级钩子的非零退出问题，避免安装完成后显示误导性错误。
- 完成亮色、暗色以及多种窗口宽度下的布局验证。

## 0.1.1-r4

- 使用 SVG 命名空间创建趋势曲线，修复只有图例而没有曲线的问题。
- 修正页面右上角工具栏在不同 LuCI 主题中的位置。
- 修正主机表格表头与数据列错位。
- 下载数据保持蓝色，上传数据保持绿色。
- 进一步改善缩放时的自适应排版。

## 0.1.1-r3

- 增加暗色主题专用的中性灰配色，降低纯黑、纯白交替造成的视觉刺激。
- 隔离插件样式，减少不同 LuCI 主题对页面的覆盖。
- 使用弹性布局、网格布局和响应式断点适配不同主题、分辨率与浏览器缩放比例。
- 改善卡片、表格、连接详情在窄屏下的可读性。

## 0.1.1-r2

- 增加可直接查看实时数据的“流量洞察”状态页面。
- 展示实时上传/下载、累计流量、主机数量、连接数量与最高风险。
- 展示内网主机列表和当前连接详情。
- 增加 ImmortalWrt 24.10.x / OpenWrt opkg 所需的 x86_64 IPK 构建与安装支持。

## 0.1.1-r1

- 首个 ImmortalWrt 25.12.0 x86_64 APK v3 安装包。
- 按内网主机累计上传、下载流量，并提供实时速率采集。
- 采集 conntrack 连接的源 IP、目标 IP、端口和方向。
- 支持离线国家/地区与 ASN 归属识别。
- 使用 GitHub 公开威胁数据集生成可解释的 0–100 IP 风险分。
- 提供 LuCI 配置页、UCI 配置、rpcd ACL 和 procd 服务。
